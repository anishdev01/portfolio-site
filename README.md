## Deploy to Vercel in 3 Steps

### Option 1 — GitHub + Vercel (Recommended)

1. Create a new GitHub repository (e.g. `portfolio`)
2. Upload `index.html` and `vercel.json` to the repo
3. Go to [vercel.com](https://vercel.com) → **New Project** → Import your GitHub repo
4. Click **Deploy** — done! ✓

### Option 2 — Vercel CLI

```bash
npm install -g vercel
cd portfolio-anish
vercel
```
Follow the prompts. Your site will be live in ~30 seconds.

### Option 3 — Vercel Dashboard Drag & Drop

1. Go to [vercel.com/new](https://vercel.com/new)
2. Drag the `portfolio-anish` folder into the deploy zone
3. Done ✓
